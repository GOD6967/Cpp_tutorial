## Getting started

### Requirements

- A C++ compiler (**g++**, **clang++**, or **MSVC**)
- Optional: [CMake](https://cmake.org/) for larger examples

### Clone

```bash
git clone https://github.com/GOD6967/C-_tutorial.git
cd C-_tutorial
```

### Compile and run an example

```bash
g++ -std=c++17 path/to/file.cpp -o program
./program
```

On Windows (PowerShell):

```powershell
g++ -std=c++17 path\to\file.cpp -o program.exe
.\program.exe
```

## Suggested learning path

1. **Setup** — install a compiler and write a simple `Hello, World`
2. **Core syntax** — variables, operators, loops, conditionals
3. **Functions** — parameters, return values, overloading
4. **Data structures** — arrays, strings, vectors
5. **Pointers & references** — how memory works in C++
6. **Object-oriented C++** — classes, constructors, inheritance, polymorphism
7. **Standard Library** — STL containers and algorithms
8. **Practice** — solve problems and build small projects
